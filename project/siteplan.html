Site Plan

Smart Used Car Buying Guide

Site Name

Smart Used Car Buying Guide

I chose this name because the website is focused on helping users make smart and safe decisions when buying a used car. The title is clear, practical, and directly connected to the purpose of the site.

Optional domain availability: smartusedcarbuyingguide.com

Site Purpose

The purpose of this website is to help people inspect, compare, and safely purchase used cars. The site will provide useful information such as a checklist for car inspections, common mechanical problems, negotiation tips, and a simple calculator to estimate costs.

Scenarios

  • What should I inspect before buying a used car from a private seller?
  • How can I know if the used car price is fair?
  • What common problems should I check before making a final decision?

Color Schema

The website will use the following colors:

  • #1D3557 - Used for headings, header, footer, and important sections.
  • #E63946 - Used for accents, buttons, highlights, and emphasis.
  • #F1FAEE - Used for the page background and a clean visual style.

Typography

  • Montserrat - Used for headings and section titles.
  • Roboto - Used for body text and general content.
